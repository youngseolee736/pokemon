# Proposal

## What I'm building
I am building a web app that lets a user upload a selfie and receive the 5 Pokémon whose facial features feel closest to their face.

## Which API I'm using
- PokéAPI: https://pokeapi.co/
- OpenAI Vision API: https://platform.openai.com/

## Why I chose this
I wanted to combine live Pokémon data with simple image analysis so the app feels like a fun, interactive real-world data experience.

## Core features
- Upload a selfie image
- Fetch Pokémon metadata and artwork from an external API
- Analyze the face and rank the top 5 closest Pokémon
- Show the results in a responsive card layout

## What I don't know yet
I am still learning how to safely route the OpenAI API key server-side and how to shape the backend response into a predictable JSON format.
